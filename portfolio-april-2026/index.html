03.3
Salesforce MCP Connector
In Build
The infra unlock

Live CRM → every AI tool.

A custom MCP (Model Context Protocol) server builds a live connection between Salesforce and every Fourth AI tool. Once live, any AI tool in the Fourth ecosystem can query Salesforce data in real time — account status, deal stage, contact history — with no manual export or copy-paste. This unblocks GTM Risk V2, Sales Brain, and any AI workflow that needs live CRM context.

Infrastructure
Azure · agreed with Danny Shine

Two dedicated servers being built in the Azure subscription. Two weeks of IT/Security alignment closed this week.

Authentication
Microsoft Entra ID · token exchange

Confirmed as the auth pattern. Every AI tool authenticates to Salesforce through the same enterprise identity.

What this unblocks
GTM Risk V2 · Sales Brain · beyond

Every AI workflow that needs live CRM context moves from "blocked" to "shipping" once this lands.
